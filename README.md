# 번개장터 클론코딩

## 💛프로젝트 설명

- React, Spring을 기반으로 번개장터 웹 사이트 클론코딩

- [번개장터 사이트](https://m.bunjang.co.kr/)

- 프론트 / 백엔드 레포지토리 분리하여 작업

- 프론트 배포 X 

- 구현 기능

  로그인/회원가입

  CR 구현(메인화면 상품 불러오기, 상품 추가하기)
